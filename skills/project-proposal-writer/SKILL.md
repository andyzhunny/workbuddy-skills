## When to Use

用户需要撰写文化类社科项目申报书，特别是：
- 地方文化研究项目（诗词、民俗、非遗等）
- 文化翻译/传播类项目
- 需要引用国家/省/市三级政策话语背书的学术项目
- 要求输出标准格式Word文档（.docx）

## Core Rules

1. **必须收集完整输入信息**：项目名称、研究对象、目标地域、文化类型、政策依据、项目周期、申报单位、研究团队、绩效指标。

2. **严格遵循四段式结构**：研究意义与目的 → 文献综述 → 内容与方法 → 绩效考核。

3. **文风必须官方学术化**：长句排比、四字词组、政策话语织入正文，严禁口语化。

4. **每个论点三层递进**：政策依据 → 学术价值 → 实践意义。

5. **创新点固定为三方面**：内容创新、视角创新、实践创新。

6. **研究方法固定七种**：文献研究法、考证法、文本分析法、跨学科研究法、实地调研法、翻译实践法、案例研究法。

7. **输出前必须调用minimax-docx**：生成格式正确的.docx文件，不得仅返回文本。
