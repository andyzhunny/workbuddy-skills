# 几何艺术风格笔记 🎨

喜欢 **Playful Geometric** 风格笔记的简约美感，灵感来自 Memphis 设计流派。

> 结合几何图形与柔和色调，创造出既现代又富有艺术感的视觉语言，让人眼前一亮。

## 核心元素

- 三角形、圆形、方形等基础几何形状的巧妙组合
- 柔和的渐变色彩与留白处理
- 简洁的线条与抽象图案

---

# 工业风笔记风格 🏭

**Neo-Brutalism** 风格笔记：RAW. LOUD. UNAPOLOGETIC.

> 大胆的色彩碰撞与粗犷的字体设计，展现出原始而强烈的视觉冲击力，拒绝平庸。

## 设计特点

- 5px 左右的粗边框设计
- 8-10px 的阴影效果
- 高对比度的色彩搭配

---

# 植物插画风格笔记 🌿

**Botanical** 风格笔记将自然元素融入设计，营造清新治愈的氛围。

> 将植物插画与文字内容有机结合，创造出既美观又实用的视觉体验

## 搭配建议

- 绿色系或大地色系的配色方案
- 手绘风格的植物插画元素
- 自然的纹理与纸张质感

---

# 多种风格选择指南

了解不同风格笔记的特点与适用场景：

1. **Professional** - 商务、正式场合的笔记风格
2. **Retro** - 复古怀旧的经典设计风格
3. **Terminal** - 程序员、极客风格的代码界面设计
4. **Sketch** - 手绘草图风格的随意笔记

使用方法示例：
```bash
python scripts/render_xhs.py example.md -t playful-geometric
```
