# ljg-translation-plan

## 技能说明

本技能专为医学翻译团队设计，用于翻译项目正式启动前的系统化规划。通过五阶段引导式对话，帮助项目经理和译者系统梳理项目背景、评估专业难度、识别高风险内容、制定分工方案，最终输出一份可直接执行的项目计划文档。

适用于药品说明书、临床试验报告、医疗器械文档、医学期刊论文等多种医学文本类型的翻译项目。

---

## 使用流程（五阶段）

### 第一阶段：项目背景分析

请提供源文本或描述这个翻译项目的基本情况。

**源文本类型是什么？** 药品说明书 / 临床试验报告 / 医疗器械文档 / 医学期刊论文 / 其他

**翻译方向是？** 中→英 / 英→中

**目标受众是？** 专业医生 / 受过专业训练的从业者 / 普通患者 / 监管机构 / 学术期刊

**发布场景是？** 上市申报 / 临床使用 / 学术发表 / 内部培训 / 其他

（每个问题依次询问，用户回答后记录信息，继续下一问题）

---

### 第二阶段：专业维度评估

**专业术语密度如何？是否已有现成的术语库或术语表？** 极高（已有完整术语库）/ 较高（部分术语可参考）/ 一般（需要大量新建）/ 较低（以通用医学词汇为主）

**本项目是否需要满足特定监管要求？** FDA / EMA / NMPA / ICH / 其他监管机构 / 无特定监管要求

**目标读者的医学素养分层是怎样的？** 专家级（可接受未经解释的专业术语）/ 从业者级（需要部分术语解释）/ 普通公众级（需要全面简化）

**是否有文化敏感内容需要特别处理？** 是（涉及宗教/地域/族群差异）/ 否 / 不确定

**文本中图表、数据、参考文献的占比如何？** 高（含大量图表和原始数据）/ 中（含部分图表）/ 低（以正文叙述为主）

---

### 第三阶段：难点预判与策略

基于前两阶段的回答，系统识别并输出以下内容：

**高风险内容识别：**
- 可能存在长难句的段落类型（如临床试验方法描述、药品作用机制说明）
- 需重点关注的监管敏感表述（如适应症、禁忌症、不良反应描述）
- 专业缩写词（如 AE、SAE、QoL、MedDRA、ICH）需要确认统一译法
- 一词多义风险词（如"administration"在医学语境中意为"给药"而非"管理"）

**中风险内容识别：**
- 需要统一但相对常见的医学术语
- 参考文献格式处理
- 剂量和单位的表达规范

**低风险内容识别：**
- 通用过渡语句和格式文本
- 简单描述性段落

**推荐的参考资源方向：**
- WHO 国际非专利名称（INN）药品译名对照
- NMPA 药品说明书模板及规范
- FDA 药品标签指南和 ICH E3 临床研究报告格式要求
- 相关治疗领域已上市药品的中英文说明书（平行文本）
- 权威医学词典和专业术语库

---

### 第四阶段：团队分工与时间规划

**文本总字数或段落数大约是多少？** 请提供估算或精确数字

**团队规模和成员专长是怎样的？** 例如：项目经理 1 人、专业医学译者 2 人、审校 1 人

**客户或内部截止日期是什么时候？** 是否有分阶段里程碑要求

**是否有往期参考译本或已有术语库可复用？** 有（请说明来源）/ 无

根据上述信息，我将给出以下建议：

**分工方案建议：**
- 术语组职责和负责人
- 初译组职责和负责人（建议按治疗领域或文档章节分配）
- 交叉审校组安排
- 终审人员指定

**里程碑节点建议：**
- 术语确认截止日期
- 初译完成截止日期
- 交叉审校截止日期
- 终审截止日期
- 客户交付日期

**时间分配比例参考：** 术语确认 10% / 初译 50% / 交叉审校 25% / 终审 15%

---

- 术语一致性要求：[基于密度和是否有现成术语库]
- 监管合规要求：[FDA / EMA / NMPA / ICH / 其他]
- 受众医学素养：[专家级 / 从业者级 / 普通公众级]
