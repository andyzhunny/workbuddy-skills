# Questioning Guide — 天赋挖掘机

## 4 个必问问题

### Q1: 童年原矿（16岁之前）

**核心意图**：在社会规训之前，人的本能倾向最真实。

**引导语**：
> "我想先带你回到一个特别的地方——16岁之前。那时候你可能还没怎么被'应该做什么'所塑造。回忆一下，有没有什么事情，是**没人逼你**，你自己就会废寝忘食去做的？不需要夸奖，不需要报酬，就是停不下来。"
> 
> "另外，有没有什么从小到大被反复批评的'顽固缺点'？比如爱插嘴、太敏感、爱发呆、太固执……这些被贴标签的东西，有时候恰好是天赋的另一种面貌。"

**追问方向**：
- "你当时做这件事的时候，是什么感觉？能描述一下吗？"
- "这件事后来为什么停了？是谁让你觉得'这不合适'的？"
- "如果没有人说你不该做这件事，你觉得你会做到什么程度？"

**信号标记**：
- 做的时候**忘我/失去时间感** → 心流信号
- 被批评后感到**委屈但无法放弃** → 天赋被压抑信号
- 多个"缺点"指向**同一底层特质** → 核心天赋候选

---

### Q2: 无意识胜任区

**核心意图**：你觉得理所当然的事，别人觉得很了不起。这就是天赋。

**引导语**：
> "现在我们来看一个成年后的信号。有没有什么事情，让你觉得'这还需要学吗？这不是显而易见的吗？'——但周围人却觉得很难、或者很佩服你能做到？"

**追问方向**：
- "你是从什么时候发现自己能做到这个的？有没有一个'啊，原来别人做不到'的时刻？"
- "当你帮别人做这件事的时候，你是怎么教的？能不能把你脑子里的步骤说出来？"
- "做这件事的时候，你觉得累吗？是身体累还是心累？"

**信号标记**：
- 步骤**无法拆解**（因为太自然） → 深层内化能力
- 教别人时感到**不耐烦**（不是故意，是真的觉得"怎么会不懂"） → 无意识胜任信号
- 别人反应"你怎么能___" → 外部确认信号

---

### Q3: 能量回血点

**核心意图**：区分"擅长但耗能"和"天赋所在"。真正的天赋做完会回血。

**引导语**：
> "这个问题很关键。回忆一下，最近有没有哪件事，做完之后虽然身体可能很累，但精神状态是**亢奋的、充了电的、停不下来的**？不是那种'终于完成了松了一口气'的疲惫，而是'我还想做、我脑子在转'的那种兴奋。"

**追问方向**：
- "这种亢奋能持续多久？是做完就消了，还是接下来几天都在想这件事？"
- "如果这件事没有人看见、没有人夸你、甚至没有报酬，你还愿意做吗？"
- "你生活中最'满'的时候——那种灵魂都充盈的感觉——是在做什么？"

**信号标记**：
- 做完**无法停止思考** → 天赋燃烧信号
- 不需要**外部认可**就愿意做 → 内在动机信号
- 身体疲惫但**精神饱满** → 回血信号 ✅（vs 单纯耗能 ❌）

### Q4: 嫉妒雷达（阴影即宝藏）

**核心意图**：嫉妒是"被压抑的天赋"在发出信号。这需要勇气，但最有价值。

**引导语**：
> "这个问题可能让你有点不舒服，但我保证它非常关键。你曾经对谁——或者对哪种生活状态——产生过强烈的**嫉妒**？不一定是那种恶意的嫉妒，可以是那种'酸溜溜的'、'我也想要但是不敢承认'的感觉。"

**追问方向**：
- "你嫉妒的是他拥有的**什么东西**？是自由？影响力？创造力？还是某种你看不见但感受到的氛围？"
- "如果深入看，你嫉妒的真的是那个'东西'，还是你觉得自己**本应该拥有却没拥有**的那个可能性？"
- "如果明天你可以拥有他生命中让你嫉妒的那个部分，但必须放弃你现有的一样东西，你愿意用什么换？"

**信号标记**：
- 嫉妒指向**某种能力或状态**（而非物质） → 天赋饥渴信号
- 嫉妒触发**愤怒或羞耻** → 被压抑的深层需求
- 多次嫉妒指向**同一方向** → 核心天赋方向

## 追加问题池（根据对话需要使用，总问题数 ≤ 10）

### Q5: 能量吸血点（反面信号）
> "反过来问——什么明明你做得不错、甚至被认可，但做完之后就像**被抽干了一样**？能分辨出是哪种'累'吗？"

### Q6: 最佳状态回忆
> "你这辈子有过几次'我是对的'的笃定时刻？不是被别人告诉的，是你内心深处100%确定的——'这就是我应该做的事'。"

### Q7: 做得最差但最开心的事
> "有没有什么事情你**做得并不好**，但就是很喜欢做？那种'我知道我做得一般但我就是享受'的感觉？"

### Q8: 别人找你最多的原因
> "如果你问身边最近的人——'你们找我帮忙最多是因为什么？'你觉得他们会说什么？不是因为你职位高，而是因为你这个人本身。"

### Q9: 时间消失瞬间
> "你能回忆起最近一次**完全忘记看手机**的时刻吗？你在做什么？持续了多久？"

### Q10: 如果不会被评判
> "如果全世界的人都看不见你在做什么，也永远不会评价你做得好不好——你会花大部分时间做什么？"

## 对话节奏控制

### 每轮对话的模板
```
[简短共鸣/反馈上一轮] → [自然过渡] → [提出一个新问题] → [留出空间]
```

### 反馈模板
- **正面**："我注意到你说___，这个细节很有意思。因为___"
- **阴影捕捉**："你说___的时候，我注意到___。你有没有觉得这个回答里有一点点矛盾？"
- **深入**："你刚才说的___，能不能再往深走一步？当你___的时候，你心里真正在想什么？"

### 何时追问 vs 何时进入下一题
- 用户回答**简短/回避** → 追问（至少2轮）
- 用户回答**丰富/有故事** → 简短反馈后进入下一题
- 用户出现**情绪波动**（停顿、长文、防御性） → 暂停提问，先接住情绪
- 信号**已经清晰** → 不必过度挖掘，进入下一题
