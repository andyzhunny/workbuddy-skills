# 06 - 完整时间线

## 核心定位
许渊冲1921年4月18日出生于江西南昌，2021年6月17日逝世，享年100岁。这条时间线纵贯一个世纪，记录了一个翻译家从求学到成为"诗译英法唯一人"的完整生命历程。

## 完整年表

### 童年与求学期（1921-1938）

| 时间 | 事件 | 意义 |
|------|------|------|
| **1921.04.18** | 出生于江西南昌 | 母亲受过新式教育，擅长绘画，赋予其爱好文学和追求美的天性 |
| **1921** | 表叔熊式一翻译《王宝钏》在英国引起轰动 | 翻译世家的基因暗示 |
| **1932** | 就读于江西省立南昌第二中学 | 基础教育 |
| **1937** | 抗日战争爆发，16岁只身赴云南 | 人生第一次重大决策 |
| **1938** | 以**第7名**成绩考入**西南联合大学**外语系 | 人生最关键转折：接触钱锺书、闻一多、吴宓等同窗名师；与杨振宁同桌 |

### 青年时期：求学与初恋（1938-1950）

| 时间 | 事件 | 意义 |
|------|------|------|
| **1939** | 大一，将林徽因《别丢掉》译成英文写成情书 | 翻译处女作，也是人生第一首"情诗译作" |
| **1941** | 应征在美国志愿空军（飞虎队）任英文翻译 | 完成"第一次中美文化沟通"——将三民主义译为林肯名言 |
| **1943** | 从西南联大毕业；在昆明天祥中学任英文教师 | 边教学边翻译 |
| **1944** | 考入**清华大学研究院**外国文学研究所，研究莎士比亚和德莱顿 | 系统西方文学学术训练 |
| **1948** | 赴法国**巴黎大学**留学，研究拉辛与莎士比亚 | 掌握英法双语能力 |
| **1950.11.07** | 获巴黎大学文学研究院文凭；从伦敦乘船回国 | 放弃法国优厚待遇，回国奉献 |

### 早期职业生涯（1950-1970）

| 时间 | 事件 | 意义 |
|------|------|------|
| **1951** | 任北京外国语学院教师 | 回国后第一份正式教职 |
| **1952** | 调入**解放军外国语学院**任教 | 军队院校任教（讲授英、法文与翻译） |
| **1950年代** | 将《西厢记》《牡丹亭》《长生殿》《桃花扇》四大诗剧译成英文 | 早期重大翻译工程 |
| **1958** | 开始将**毛泽东诗词**译成英文与法文 | 开创性工作，成为英法韵文翻译唯一专家的开始 |
| **1959.除夕** | 在北京欧美同学会舞会上邂逅夫人**照君**，一见钟情 | 相濡以沫60年的伴侣 |

### 中年：动荡与坚持（1960-1983）

| 时间 | 事件 | 意义 |
|------|------|------|
| **1960-1970** | 随香山外国语学校迁至**张家口**任教 | "文革"前后的动荡期 |
| **1966-1976** | **文革**——最大的人生磨难：因翻译毛泽东诗词被批"歪曲毛泽东思想"，挨100鞭子 | 翻译成为生命支柱："我一译诗，就把热、累、批、斗全都忘到九霄云外去了。" |
| **1970-1983** | 在**洛阳外国语学院**任教（讲授英法文与翻译） | 教学积累期 |
| **1978** | 开始发表翻译理论文章（至1983年共12篇） | 理论建构期的开始 |
| **1983** | 任**北京大学**国际政治系教授；后转入北大国际文化教研室（后并入新闻传播学院） | 回到学术自由环境，人生最丰产阶段的起点 |

### 黄金期：巅峰岁月（1984-2000）

| 时间 | 事件 | 意义 |
|------|------|------|
| **1984** | 在国际关系学院开设中西文化比较课程 | 学术实践 |
| **1984** | 出版《翻译的艺术》（论文集，1978-1983年12篇理论文章） | 理论体系初成 |
| **1989** | 翻译出版《红与黑》，引发翻译界大讨论 | 最具争议性译作 |
| **1991** | **70岁正式退休** | 人生转折：退休后迎来翻译巅峰 |
| **1992-1995** | 《中国翻译》不刊发其文章（王佐良压下其博士生导师申请） | 学术压制期 |
| **1996** | 出版《追忆逝水年华》（三联书店） | 人生回顾与文学总结 |
| **1999** | 被十所大学教授联合提名为**诺贝尔文学奖候选人** | 国际认可的标志 |
| **2000** | 出版《唐诗三百首》（高等教育出版社） | 代表性工程完成 |

### 晚年：荣耀与冲刺（2000-2021）

| 时间 | 事件 | 意义 |
|------|------|------|
| **2004** | 出版《译笔生花》，首次提出"中国学派的文学翻译理论" | 理论体系成熟 |
| **2005** | 出版《论语》《许渊冲英译毛泽东诗词》 | 经典译作持续出版 |
| **2007** | 患直肠癌，医生预估最多还有7年寿命 | "死缓"判决 |
| **2009** | 出版《楚辞》（被美国学者誉为"英美文学领域的一座高峰"）；出版《元曲三百首》 | 楚辞英译获极高评价 |
| **2010** | 获**中国翻译文化终身成就奖** | 中国翻译界最高荣誉之一 |
| **2011** | 清华大学设宴庆祝90岁生日 | 与杨振宁等老友重聚 |
| **2013** | 出版27册《许渊冲全集》（北京图书博览会首发） | 毕生译著集大成 |
| **2014.08.02** | 获国际译联**"北极光"杰出文学翻译奖**（93岁） | 国际翻译界最高奖，首位亚洲获奖者 |
| **2017** | 登上央视《朗读者》第一季压轴（96岁），朗诵1939年译作《别丢掉》 | "出圈"，成为公众人物 |
| **2018** | 相濡以沫60年的夫人照君去世 | 晚年最大损失 |
| **2021.04.18** | 北京大学举办"许渊冲先生翻译思想与成就研讨会"（百岁寿辰） | 生前最后一场学术活动 |
| **2021.06.17.07:40** | 于北京家中安详逝世，享年100岁 | 百年翻译人生画上句号 |
| **2021.06.22** | 遗体告别仪式于八宝山殡仪馆大礼堂举行 | 盖棺定论 |

## 核心时间节点分析

### 人生三大转折点

1. **1938年（17岁）**：考入西南联大 → 接触到钱锺书等大师，建立翻译志向
2. **1983年（62岁）**：进入北大 → 迎来翻译创作高峰期（退休后30年产出远超前半生）
3. **2007年（86岁）**：患癌后被续命14年 → "好像上天的一场暗示"，开始冲刺莎士比亚全集翻译

### 人生规律

- **播种期**（0-30岁）：求学积累
- **蛰伏期**（30-60岁）：教学+动荡，翻译能力在夹缝中成长
- **绽放期**（60-100岁）：退休后的"自由王国"，迎来人生最丰产时期

> 许渊冲自己对人生的总结："生命不是你活了多少日子，而是你记住了多少日子，要让你生命里的每一个日子都值得记忆。"
